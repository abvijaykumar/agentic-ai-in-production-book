# Google Colab Notebooks

This directory contains the **Google Colab notebooks** for the book *"Agentic AI in Production"*.

These notebooks provide interactive, hands-on examples for the concepts covered in each chapter. They are designed to run in the cloud, allowing you to experiment with Agentic AI frameworks and libraries without complex local setup.

## Usage
Each notebook corresponds to a specific chapter or concept in the book. Open them in Google Colab to run the code, visualizes the agents, and modify the examples to deepen your understanding.
