# Agentic AI in Production - Book Code

Welcome to the official source code repository for the book **"Agentic AI in Production"**. This repository contains all the code examples, projects, and notebooks referenced throughout the book.

The repository is organized into two main directories:

## 1. collabs
This folder contains **Google Colab notebooks** corresponding to the chapters in the book. These notebooks are designed to provide interactive, hands-on examples of the concepts covered in each respective chapter.

## 2. supportmax-pro
This folder hosts the **SupportMax Pro** application, which serves as the primary enterprise use case for the book. You will find various versions of the application here, demonstrating its evolution and improvement as we apply more advanced Agentic AI concepts.

### Application Versions:
- **v0.5-baseline**: The initial baseline version of the application.
- **v1-mvp**: The Minimum Viable Product implementation.
- **v2-cognitive**: Enhanced version with cognitive capabilities.
- **v3-enterprise**: The fully realized enterprise-grade application.

Each version folder represents a step forward in the application's lifecycle, reflecting the progression of topics discussed in the book.
