# SupportMax Pro Documentation

## API Rate Limits
The SupportMax Pro API has a default rate limit of 100 requests per minute per user.
To increase this limit, please contact sales@supportmax.com.

## Troubleshooting 500 Errors
If you encounter a 500 Internal Server Error:
1. Check the server logs for stack traces.
2. Ensure your API key is valid.
3. Verify that the payload size does not exceed 10MB.

## Account Lockout Policy
Accounts are locked after 5 failed login attempts.
To unlock an account, the user must reset their password via email or contact an administrator.
